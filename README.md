# COMP3123 Lab Test 2 - Weather App

## Description
A simple React weather app that fetches real-time current weather data by city search and displays temperature, conditions, and an icon.

## API Used
OpenWeatherMap Current Weather API  
Endpoint: https://api.openweathermap.org/data/2.5/weather

## Setup Steps
1. Install dependencies:
   - npm install
2. Create a `.env` file in the project root:
   - REACT_APP_OWM_KEY=YOUR_OPENWEATHERMAP_KEY
3. Run the app:
   - npm start

## Features
- Search by city 
- Displays temperature, feels-like, min/max, humidity, pressure, wind, clouds, visibility
- Weather icon and themed UI
